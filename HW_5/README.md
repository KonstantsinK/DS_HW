Набор данных о ценах на ноутбуки.
Этот набор данных представляет собой набор функций, связанных с различными ноутбуками, такими как бренд, тип процессора, оперативная память, емкость хранения и другие спецификации. В набор данных также включены соответствующие цены на эти ноутбуки. 
Набор данных содержит 823 строки и 19 столбцов.
Все значения ненулевые.
Тип данных большинства столбцов — категориальный.
Цена, количество оценок и количество отзывов указаны в числовом формате.

 Для классификации использовались следующие модели:
K Neighbors Regressor
Decision Tree
Random Forest Regressor
Adaboost Regressor
Gradient Boost Regressor 
LightGBM
 Гипотезы:
- есть некие опредленные характеристики ноутбука, которые вляиют на ценообрзование
- цену ноутбука можно спрогнозировать на основе данных о его характеристиках
