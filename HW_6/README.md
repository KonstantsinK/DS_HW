Цена на сырую нефть

Ежемесячная цена на сырую нефть с 1983 года по настоящее время.

Набор данных по сырой нефти WTI (USD/Bbl) был создан с расчетом на то, чтобы понять влияние мировых цен на нефть на экономику любой страны.

Источник

Используется https://tradingeconomics.com/commodity/crude-oil, этот набор данных создается и обновляется ежемесячно.

Анализ временного ряда производился с использованием:

модели ARIMA,

модели SARIMA,

модели Холта-Уинтерса

LSTM

автоэнкодера
