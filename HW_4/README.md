Набор о сотрудниках.

Предназначен для HR-аналитики, планирования рабочей силы и создания моделей машинного обучения для прогнозирования оттока / истощения сотрудников.

Содержит 15,000 записей сотрудников для HR-аналитики и прогнозирования оттока. Каждая запись включает в себя:

    EmpId : Уникальный идентификатор сотрудника.
    
    Удовлетворенность: оценка удовлетворенности работой (от 0 до 10).
    
    Оценка: оценка служебной аттестации (0 - 10).
    
    number_of_projects : Общее количество завершенных проектов.
    
    средний_монтly_hours : Среднее количество рабочих часов в месяц.
    
    time_spent_company : Количество лет, проведенных в компании.
    
    job_Accident : Указывает, если у работника произошел несчастный случай, связанный с работой (1 = Да, 0 = Нет).
    
    Повышение : Повышен ли работник был повышен в должности в последние 5 лет (1 = Да, 0 = Нет).
    
    Название отдела (например, продажи, техническое обслуживание, поддержка).
    
    Salary_INR : Годовой оклад работника в INR.
    
    Отток: Целевая колонка; 1 = работник ушел, 0 = сотрудник остался.
    
 Для классификации использовались следующие модели:
 
CatBoost,
GradientBoosting,
KNeighbors,
RandomForest,
LogisticRegression,
StackingClassifier с базовыми классификаторами - CatBoost, RandomForest, ExtraTrees и мета-классификатором LogisticRegression.
