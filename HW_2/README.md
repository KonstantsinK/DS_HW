Набор данных о бронировании отелей.
Контекст
Онлайн-каналы бронирования отелей резко изменили возможности бронирования и поведение клиентов. Значительное количество бронирований отелей отменяется из-за отмены или неявки. Типичные причины отмены включают изменение планов, конфликты планирования и пр.
